# Fizz Layout

A flexible Vue 3 layout component library built with Element Plus.

## Installation

```bash
npm install fizz-layout
# or
pnpm add fizz-layout
```

## Peer Dependencies

```bash
pnpm add vue vue-router element-plus @vueuse/core @iconify/vue
# Optional for i18n support
pnpm add vue-i18n
```

## Usage

```vue
<script setup>
import { FizzLayout } from 'fizz-layout'
import 'fizz-layout/style.css'
</script>

<template>
  <FizzLayout />
</template>
```
